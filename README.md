# Chatbot Project (React + Vite)

A simple and clean chatbot UI built using **React**, **Vite**, and modern JavaScript.  
The project demonstrates basic message handling, component structure, and UI layout for a chatbot interface.

---

##  Features

- Send & display chat messages  
- Bot and user message styling  
- Avatar images for user and bot  
- Fast development with Vite  
- Clean and minimal UI  
- Easy to extend with APIs or AI models
